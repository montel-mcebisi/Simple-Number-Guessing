A simple Java number guessing game where the program generates a random number between 0 and 9 and gives the player three attempts to guess it. After each guess, the user receives feedback indicating whether their guess is too high or too low. The game ends when the correct number is guessed or all attempts are exhausted, resulting in a loss.
Features:
Generates a random number between 0 and 9
Allows the player 3 attempts to guess the number
Provides feedback after each guess:
Too low if the guess is smaller than the target number
Too high if the guess is larger than the target number
Displays "Correct guess!" when the number is guessed successfully
Displays "You lose!" if all attempts are used without guessing the number
